## Region & autocapture scope
- **Region:** EU (`eu.i.posthog.com`). Snax must create the PostHog account in the EU region ‚Äî accounts are region-locked.
- **Autocapture ignorelist** ‚Äî disable autocapture (click/input recording) on admin, host-only, and private user routes. Explicit `posthog.capture(...)` calls still fire on these routes if added in future.

```ts
const ADMIN_ROUTES: RegExp[] = [
  /^\/underboss(\/|$)/,
  /^\/admin(\/|$)/,
  /^\/shipping(\/|$)/,
  /^\/graphics(\/|$)/,
  /^\/partner(\/|$)/,      // matches /partner and /partner/*, NOT /partner-intake/*
  /^\/host(\/|$)/,
  /^\/checkin(\/|$)/,
  /^\/dj(\/|$)/,
  /^\/post(\/|$)/,
  /^\/display(\/|$)/,
  /^\/account(\/|$)/,
  /^\/new$/,
];
```
